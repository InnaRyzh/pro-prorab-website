📁 Структура папок для фотографій

images/

  • hero/ - фотографії для головної секції
  • portfolio/ - фотографії проектів
  • about/ - фотографії про компанію
  • services/ - фотографії послуг
  • contacts/ - фотографії для контактів

🎯 ШАГ 1: Підготуйте фотографії

Рекомендації по фотографіям:

  • Формат: JPG, PNG, WebP
  • Розмір: не більше 2MB на фото
  • Якість: 1920x1080px для великих фото
  • Назви файлів: тільки англійські букви, цифри, дефіси

⚡ ШВИДКИЙ СПОСІБ ДОДАВАННЯ ФОТО

1. Створіть файл replace-images.js:

// replace-images.js const imageReplacements = { // Головна сторінка 'images/project1.jpg': 'images/portfolio/project1.jpg', 'images/project2.jpg': 'images/portfolio/project2.jpg', 'images/project3.jpg': 'images/portfolio/project3.jpg', 'images/project4.jpg': 'images/portfolio/project4.jpg', // Hero фон 'images/hero-bg.jpg': 'images/hero/hero-bg.jpg', // Про нас 'Фото команди Pro Prorab': 'images/about/team-photo.png', // Послуги 'Дизайн інтер\'єру': 'images/services/interior-design.jpg', 'Будівельні роботи': 'images/services/construction.jpg', 'Авторський нагляд': 'images/services/supervision.jpg', 'Комплектація об\'єктів': 'images/services/furnishing.jpg' }; // Автоматична заміна document.addEventListener('DOMContentLoaded', function() { // Заміна src у img тегів document.querySelectorAll('img').forEach(img => { const currentSrc = img.getAttribute('src'); if (imageReplacements[currentSrc]) { img.setAttribute('src', imageReplacements[currentSrc]); } }); // Заміна фонових зображень document.querySelectorAll('[style*="background-image"]').forEach(el => { const style = el.getAttribute('style'); Object.keys(imageReplacements).forEach(oldPath => { if (style.includes(oldPath)) { const newStyle = style.replace(oldPath, imageReplacements[oldPath]); el.setAttribute('style', newStyle); } }); }); });
