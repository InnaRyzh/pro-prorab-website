Portfolio images structure
==========================

Add your project photos into these folders:

- images/portfolio/lakewood/
- images/portfolio/great/

File naming
-----------
- Use sequential names starting from 1.jpg: 1.jpg, 2.jpg, 3.jpg, ...
- Recommended: JPG format, sRGB, 1600–2400 px on the long side.
- The grid preview uses 1.jpg from each folder.

How many images?
----------------
- The site is currently set up for three images per project.
- To add or change the list, edit js/script.js, object `galleryMap` (arrays under `lakewood.images` and `great.images`).

Notes
-----
- If an image is missing, the site shows a temporary Unsplash placeholder.
- You can rename the projects later; this file is just a guide.
